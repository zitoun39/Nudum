# Security Policy for Nudum Platform

## Supported Versions

Only the latest active release on `main` is supported with security updates.

| Version | Supported |
| ------- | --------- |
| 1.0.x   | Yes       |

## Reporting a Vulnerability

We take the security of Nudum seriously. If you find any vulnerability, please do NOT create a public issue. Instead, report it directly to the lead maintainer:

- Email: abdelhak.zitoun39@gmail.com
- Expected response time: Within 48 hours

Please include:

- A description of the vulnerability.
- Steps or scripts to reproduce the issue.
- Potential impact.
